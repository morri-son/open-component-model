# Sigstore Integration Tests

Integration tests for the OCM sigstore signing/verification handler.
Tests cover key-based signing, keyless (Fulcio + OIDC), Rekor v1/v2, TSA, TUF, and minimal bundles.

## Prerequisites

- [kind](https://kind.sigs.k8s.io/)
- kubectl
- [Helm](https://helm.sh/)
- curl, openssl, xxd, shasum, jq

## Running

```bash
# 1. Create Kind cluster with Sigstore stack (Rekor v1+v2, Fulcio, TSA, TUF)
task bindings/go/sigstore/integration:kind/setup

# 2. Run integration tests (env vars loaded automatically from hack/generated.env)
task bindings/go/sigstore/integration:test/integration

# 3. Tear down
task bindings/go/sigstore/integration:kind/teardown
```

The setup script writes `hack/generated.env` with all required `SIGSTORE_*` environment variables.
The `test/integration` task loads this file automatically via `dotenv:`, so no manual export is needed.

## What the setup deploys

| Component | Source | Local port |
|-----------|--------|------------|
| Fulcio | sigstore/scaffold Helm chart | 5555 |
| Rekor v1 | sigstore/scaffold Helm chart | 3001 |
| TSA | sigstore/scaffold Helm chart | 3002 |
| TUF | sigstore/scaffold Helm chart | 8088 |
| Rekor v2 | rekor-tiles Helm chart (POSIX backend) | 3003 |

## Test structure

Tests that require optional infrastructure skip gracefully when the corresponding
environment variable is not set:

| Test | Requires |
|------|----------|
| `KeyBased_SignVerify` | Rekor (defaults to public-good if unset) |
| `KeyBased_MinimalBundle` | Nothing (pure offline test) |
| `Keyless_SignVerify` | Fulcio + OIDC token + trusted root |
| `TSA_SignVerify` | TSA URL |
| `TUF_TrustedRoot` | TUF mirror + initial root |
| `SigningConfig` | Signing config file paths |

## Notes

- The OIDC token generated by the setup script is valid for 1 hour.
  Re-run `kind/setup` if it expires (or regenerate with `kubectl create token`).
- Rekor v2 tests run only when `SIGSTORE_REKOR_V2_URL` is set.
- Without `hack/generated.env`, tests that need local infrastructure will
  either skip or fall back to public-good Sigstore endpoints.
